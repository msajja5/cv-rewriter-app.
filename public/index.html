🎧 How to connect Zoom / Teams audio

✅ Option 1 — Mic + Tab (recommended, no screen sharing needed)

  1. Select 🎤+🖥️ Mic + Tab audio mode
  2. Click “Share tab / screen audio” or press Start Listening
  3. In the Chrome dialog, pick your Zoom / Teams tab
  4. Tick “Share tab audio” — done!

Both your voice (mic) and the interviewer’s voice (tab) are transcribed simultaneously.

🟢 Option 2 — Microphone only (simplest)

Keep 🎤 Microphone mode selected and press Space. No sharing needed — just speak and the app listens via your mic directly.

🟣 Option 3 — Browser Tab / System Audio only

  1. Select 🖥️ Browser Tab / Call or 🔊 System Audio
  2. A Deepgram API key is required for these modes (browser STT only captures mic)
  3. Click Start Listening → pick the tab or screen → tick “Share tab audio”
⚠️ Tab / System modes require Deepgram. Browser’s built-in STT (Web Speech API) can only capture your microphone — it cannot process a shared tab or system audio stream. Configure a Deepgram key in the API Keys tab to enable these modes.
